Web Development

History of Web Development

  • 1990-Tim Berners-Lee wrote the first proposal for the World Wide Web (WWW)
  • 1991The first model of HTML was invented, containing 18 HTML tags.
  • 1992-Table-based web design was introduced.

About Web Development

  • Web development refers to the creating, building, and maintaining of websites.
  • It includes aspects such as web design, web publishing, web programming, and database management.
  • It is the creation of an application that works over the internet i.e. websites.

Advantages of Web Development

  • Search Engine Optimization (SEO)
  • Adapted web design
  • More security and reliable control

Disadvantages of Web Development

  • Requires coding knowledge and learning multiple programming languages
  • Includes long hours and a high-pressure environment
  • Requires consistent changes to meet best practice guidelines and evolving technology
  • Involves dealing with naive or difficult clients
